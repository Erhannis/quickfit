OpenSCAD translation and modification of RichRap's Quick-Fit X Carriage https://www.thingiverse.com/thing:19590 (CC-A).
I've additionally converted it into a platform for the RepRap Snappy 3 https://github.com/revarbat/snappy-reprap (GPL-2).

Run get_deps.sh to get the dependencies.  (Or, if on windows, create a folder "deps.link" and in it run the git commands given and the bottom of get_deps.sh .)
